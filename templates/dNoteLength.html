⏱️ Note Length

Each note symbol has a duration—how long it lasts. Here are a few common ones:

  • Whole note (𝅝) – 4 beats
  • Half note (𝅗𝅥) – 2 beats
  • Quarter note (♩) – 1 beat
  • Eighth note (♪) – ½ beat
  • Sixteenth note (𝅘𝅥𝅯) – ¼ beat

Rests follow the same timing! For example, a quarter rest means silence for 1 beat.

Note: The number of beats depends on the time signature, which we’ll cover soon.
